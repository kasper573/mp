# api-service

A GraphQL based API service and client (client exposed as @mp/api-service/client).

- Server: [Apollo Server](https://www.apollographql.com/docs/apollo-server/) + [grats](https://grats.capt.dev/)
- Client: [Apollo Client](https://www.apollographql.com/docs/react/) + [gql.tada](https://gql-tada.0no.co/)
